# Notion Insert Icon

This extension allows you to insert custom emoji by pressing Ctrl+i in Notion.

## Usage

1. Open Notion
2. Press Ctrl+i
3. Enter the emoji name
4. Press Enter
5. The emoji will be inserted at the cursor position
